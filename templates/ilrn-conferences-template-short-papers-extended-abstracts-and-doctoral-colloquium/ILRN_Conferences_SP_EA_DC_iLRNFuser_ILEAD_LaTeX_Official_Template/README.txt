
# iLRN Conferences LaTeX Template

This project contains a custom LaTeX template for submissions to **iLRN** conferences, using the `iLRNConfProc.cls` class.

## 📄 Files Included

- `iLRNConf_template_with_instructions.tex` — Main LaTeX file with instructions and placeholder examples
- `iLRNConfProc.cls` — Custom class file that defines all formatting rules based on iLRN and Springer MathPhysSci guidelines
- `iLRN_header_logo.png` — iLRN Logo image
- `references.bib` — BIB example file
- `spmpsci.bst` — Springer's MathPhys bibliography style file
- `README.txt` — This file

## ✅ Features

- A4 paper, 10pt, Times New Roman
- 1 inch (2.54cm) margins
- Centered logo support (see class for customization)
- Formatted title and author block
- Custom abstract and keywords commands:
  ```latex
  \iLRNabstract{Your abstract text here...}
  \iLRNkeywords{Keyword1, Keyword2, Keyword3}
  ```
- LNCS-like section formatting
- Page numbers disabled

## 🚀 Getting Started

1. Edit `iLRNConf_template.tex` with your content
2. Compile the project
